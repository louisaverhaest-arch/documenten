# Documenten

Publieke documentbibliotheek via GitHub Pages.

## Nieuwe PDF toevoegen

1. Open de map `docs`.
2. Klik op **Add file** > **Upload files**.
3. Sleep je PDF in het venster.
4. Klik onderaan op **Commit changes**.
5. Na ongeveer 30 seconden verschijnt het document automatisch op de startpagina.
